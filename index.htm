How to host a website on GitHub Pages for free also using your registered domain name

Hello, World!

After being absent for ~10 years from anything related to website design and/or hosting I'm exited to be back!
Still having had registered a domain name for years, the initial question was were/how to host a simple website for free in 2024?

I figured out that GitHub via the so called GitHub Pages basically provides you a free webserver,
which can be used to upload *.HTML files to the web to be accessible via a GitHub based URL.
It is also possible to map this respective GitHub based URL to your registered domain.


Here is a short step-by-step guide how how to proceed:

1.) If not existing yet create a simple website called index.html.
This can be either done via *.TXT editor like Notepad or if you are not familiar with this topic by using a template.

2.) Open GitHub (https://github.com/) and if not existing create a GitHub account for free.

3.) Once you have logged in to GitHub under the "Start writing code" section enter a name of your website project under "Repository name *" (=for example "website").
Then on the radio buttons below select "Public" and click the "Create a new repository" button.

4.) On the next page click on the "uploading an existing file" link and uploade your "index.html" website via drag and drop.
Afterwards click the "Commit changes" button.

5.) On the top left corner select "Settings", then go to "Pages", within the "Branch" section change from "None" to "Main" and click on the "Save" button.

6.) Your website based on the uploaded "index.html" file should be available via the URL shown under "Your site is live at" section.
(Note: If you try to open the website via this URL it might take a couple of seconds until minutes until it got published accordingly!)
